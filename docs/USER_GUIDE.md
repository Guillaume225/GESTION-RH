# Guide Utilisateur

## Connexion

1. Accédez à l'application via votre navigateur.
2. Entrez votre adresse e-mail et votre mot de passe.
3. Cliquez sur **Se connecter**.

## Tableau de bord

Le tableau de bord affiche un résumé de votre organisation :
- **Total employés** : Nombre d'employés actifs
- **Congés en attente** : Demandes en attente de validation
- **Masse salariale** : Montant mensuel total
- **Postes ouverts** : Offres d'emploi actives
- Graphiques de répartition par département

## Gestion des employés

### Consulter la liste
- Utilisez la barre de recherche pour trouver un employé par nom ou prénom.
- Filtrez par département ou statut.

### Ajouter un employé (RH / Admin)
1. Cliquez sur **Ajouter un employé**.
2. Remplissez les informations : nom, prénom, e-mail, département, poste, date d'embauche, salaire, type de contrat.
3. Validez le formulaire.

## Gestion des congés

### Demander un congé
1. Allez dans **Congés** > **Demander un congé**.
2. Sélectionnez le type (Congés payés, RTT, Maladie, etc.).
3. Choisissez les dates de début et de fin.
4. Ajoutez un motif si nécessaire.
5. Soumettez votre demande.

### Approuver/Refuser (Manager / RH)
- Les demandes en attente apparaissent dans la liste des congés.
- Cliquez sur une demande pour l'approuver ou la refuser avec un commentaire.

### Solde de congés
- Consultez votre solde depuis la page des congés.

## Paie

### Consulter ses bulletins
- Accédez à la section **Paie** pour voir vos bulletins.
- Chaque bulletin détaille : salaire brut, cotisations, salaire net.

### Générer des bulletins (RH)
1. Cliquez sur **Générer les bulletins**.
2. Sélectionnez la période.
3. Validez les bulletins après vérification.

## Recrutement

### Gérer les offres d'emploi
- Créez de nouvelles offres avec le titre, département et type de contrat.
- Publiez ou clôturez les offres.

### Pipeline de candidats
- Suivez les candidats à travers les étapes de recrutement.

## Performance

### Évaluations
- Consultez les évaluations planifiées.
- Managers : Créez et complétez les évaluations de vos équipes.

### Objectifs
- Définissez des objectifs mesurables.
- Suivez la progression.

## Formation

- Consultez le catalogue de formations disponibles.
- Inscrivez-vous aux formations qui vous intéressent.
- L'indicateur de capacité montre les places restantes.

## Documents

- Accédez à vos documents (contrats, bulletins, attestations).
- Téléversez de nouveaux documents.

## Pointage

- **Pointer l'arrivée** : Enregistre votre heure d'arrivée.
- **Pointer le départ** : Enregistre votre heure de départ.
- Consultez le résumé hebdomadaire avec les heures totales et supplémentaires.

## Rapports (RH / Admin)

- **Répartition par département** : Diagramme circulaire
- **Taux de rotation** : Évolution mensuelle
- **Pyramide des âges** : Distribution par tranche d'âge
- **Taux d'absentéisme** et **Synthèse de la paie**

## Langue

Cliquez sur le bouton **EN/FR** en haut à droite pour changer la langue de l'interface.
